流／程／引／导

基础信息收集

对物资、服务类招标信息收集，包括招标批次，标段，供应商等信息通过导入模版方式收集。

唱标情况收集展示

收集供应商对每个标段、报报价情况，分类统计每个标段、报供应商报价情况，分别输出技术、商务阅标模版。

专家打分信息

根据类型导出技术、商务、价格模版，对专家信息进行分组收集，按模版收集专家收集技术部分打分情况，按模版收集专家收集商务部分打分情况。

打分情况统计分析

根据公式计算评标价格、计算得分表、计评标技术评分表、计评标价格评分表、计算综合得分、统计出最终中标人，并输出。

模版功能

商务、技术模版维护，用户维护专家打分模版，用户维护技术打分模版，用户维护商务打分模版，用户维护价格打分模版。

系统参数

公式维护，包括物资类和非物资类公式维护，权重功能维护。
